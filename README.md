Unittestaus

 - Lataa projekti
 - Avaa poetry virtuaaliympäristö
 - Varmista että ympäristössä on pytest
 - Lisää pytest tällä komenola jos ei ympäristössä "poetry add pytest --group dev"
 - Suorita testit komennolla "pytest src"
